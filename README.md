# 🌟 Professional Django Portfolio

A modern, full-stack portfolio website designed to showcase projects and services with a focus on speed and design.

## ✨ Features
- **Dynamic Services:** Manage ML, Django, and UI/UX categories via Admin.
- **Glass-morphism UI:** Built using Tailwind CSS for a premium look.
- **Contact Form:** Integrated with Django's SMTP backend for real-time emails.
- **Resume Download:** Direct download feature for recruiters.
- **Production Ready:** Configured for PostgreSQL and WhiteNoise for Render deployment.

## 🛠️ Tech Stack
- **Backend:** Django 5.0
- **Frontend:** Tailwind CSS, JavaScript, FontAwesome
- **Database:** SQLite (Local) / PostgreSQL (Production)
- **Deployment:** Render

## 🚀 Installation
 
1. Install requirements: `pip install -r requirements.txt`
2. Run migrations: `python manage.py migrate`
3. Start server: `python manage.py runserver`
